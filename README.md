# 抚柯集
柳之仪，字抚柯。

# 序
诗者，心之声也；词者，情之托也。
予生平凉，长于陇上，少慕风雅，长涉江湖。自束发执笔，迄今十载，或行于陇原，或游于蜀地，或客于京华，或怀于长安。凡所见山川风物、世态人情、悲欢离合，一一付之笔墨，录为篇章。
少年时，意气疏朗，心无尘埃，写初见则情动于中，吟风物则神驰于外。及长，奔走四方，为理想趋驰，为生计奔波，历疫情之困，经世路之艰，诗风渐趋沉郁，多苍凉感怀之音，亦存耿介不屈之气。
其间念亲恩、怀故友、思初心、忆旧人，或登高望远，或孤灯夜坐，或驰马西风，或静对狸奴，所作非为炫技，非为成名，只为记一段岁月，存一片真心。
或有人见予笔下多写风月、善记柔情，便谓予轻佻。
予之笔下，无轻薄之语，无亵渎之辞；
予之心中，无朝三之念，无暮四之情。
少年初遇，以岁月为笺，记眉眼温柔，录平生感念。
一生所重者三：一曰初心，二曰亲情，三曰故旧。
非花心，非轻佻，非浪荡，非薄幸。

集此小稿，非敢称诗，仅为十年行迹作一记，半生心事作一藏。

是为序。
柳之仪 2026年2月19日 记
